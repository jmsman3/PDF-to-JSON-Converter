# Security Policy

## Supported versions

This repository supports the latest `main` branch.

## Reporting a vulnerability

If you discover a security issue, do not open a public issue with sensitive details.

Instead, report it privately to the repository maintainer or open a minimal issue that only describes the general problem without exposing sensitive document content.

## What to include

- A short description of the issue
- The affected file type or workflow
- Steps to reproduce if they are safe to share
- Any non-sensitive logs or screenshots
