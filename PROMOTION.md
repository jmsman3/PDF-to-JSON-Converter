# Promotion Kit

Use these drafts when posting the project on LinkedIn, X, Facebook groups, Reddit, or other communities.

## Core positioning

A lightweight Python tool to convert PDF documents into structured JSON for automation, sales workflows, and data extraction.

## LinkedIn post draft

I built an open-source Python tool called **PDF-to-JSON-Converter**.

It converts PDF files into structured JSON and extracts markdown content in a simple, reusable way.

This is especially useful for teams that work with:
- sales lead PDFs
- contact directories
- vendor lists
- event attendee sheets
- inquiry forms
- business documents that need to move into CRM or automation workflows

Instead of manually copying data from PDFs, you can extract and structure it much faster.

If you work with PDF data extraction, sales operations, or document automation, check it out, star it, and share feedback.

GitHub: https://github.com/jmsman3/PDF-to-JSON-Converter

## Posting tips

- Lead with a real workflow problem, not only a technical feature.
- Mention sales, operations, or automation if you want more business interest.
- Add one screenshot or sample JSON preview if possible.
- Ask people to star, fork, or share feedback.
- Repost the same message in a few communities over time instead of only once.
